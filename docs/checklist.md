# Release Checklist

## 1. Product
- Импорт YouTube работает для лимитов MVP.
- Редактор шагов и кадров стабилен.
- Экспорт MD/PDF генерируется корректно.
- Шаринг по ссылке read-only работает.

## 2. Pipeline
- Очереди настроены, ретраи не приводят к дублям.
- TraceId проходит через все воркеры.
- Ошибки пишутся в единый формат.

## 3. Security
- Доступ к гайдам только у владельца.
- Токены шаринга имеют срок действия.
- Секреты не попадают в логи.

## 4. Infra
- Локальные модели запускаются (Whisper/Vosk, Ollama).
- Провайдеры переключаются local/remote.
- Ретенция работает (TTL джобы).

## 5. Observability
- Метрики и логи доступны.
- Алерты по `failed` и таймаутам настроены.
