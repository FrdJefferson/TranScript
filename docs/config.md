# Configuration

## 1. Провайдеры
- `asr.provider=local|remote`
- `llm.provider=local|remote`

## 2. Лимиты
- `limits.video_duration_minutes`
- `limits.concurrent_jobs_per_user`
- `limits.export_per_day`

## 3. Модели
- `asr.model`
- `llm.model`
- `llm.context_size`

## 4. Storage
- `storage.s3_endpoint`
- `storage.s3_bucket`

## 5. Feature flags
- `features.share_links`
- `features.exports`
