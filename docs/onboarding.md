# Onboarding

## 1. Подготовка окружения
- Node.js LTS
- Docker/Docker Compose
- ffmpeg

## 2. Локальный запуск
- Запустить Postgres/Redis/MinIO
- Запустить API и Worker
- (Опционально) Ollama и локальные модели

## 3. Проверка
- Импорт тестового URL
- Проверка статуса пайплайна
- Экспорт в MD

## 4. Структура репозитория
- `apps/api`
- `apps/worker`
- `apps/web`
- `packages/shared`
- `docs/`
