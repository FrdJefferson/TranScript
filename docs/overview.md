# Overview

## 1. Продукт
Сервис превращает длинные и неструктурированные видео (в первую очередь YouTube) в понятные текстовые гайды с шагами и иллюстрациями. Гайд можно читать, редактировать, сохранять и использовать повторно.

## 2. Проблема
Большая часть полезных знаний находится в формате видео, который:
- неудобно пересматривать,
- сложно быстро сканировать,
- плохо подходит для пошагового использования,
- трудно переиспользовать как документацию.

## 3. Цели MVP
- Быстрый путь: URL YouTube → гайд.
- Редактирование шагов и иллюстраций.
- Экспорт в Markdown и PDF.
- Один язык на старте (уточняется).

## 4. Нефункциональные требования
- Асинхронная обработка (очереди).
- Надежность пайплайна (ретраи, статусы).
- Прозрачный прогресс в UI.
- Масштабируемость воркеров.
- Возможность локального разворачивания ASR/LLM (Whisper/Vosk + Ollama).

## 5. Модель доступа
- MVP: приватные гайды по умолчанию.
- Доступ только владельцу, опционально — шаринг по ссылке (read-only).
- Расширение на совместную работу — отдельный этап.

## 6. Ограничения MVP
Фиксируются конфигом на старте:
- лимит длительности видео,
- набор поддерживаемых языков,
- лимит экспорта и число активных задач на пользователя.
