# Roadmap

## Q1 — MVP
- Импорт YouTube.
- ASR → шаги → кадры.
- Редактор текста и кадров.
- Экспорт MD/PDF.
- Шаринг по ссылке (read-only).

## Q2 — Качество
- Улучшение сегментации (LLM + правила).
- Улучшение подбора кадров (скоринг).
- Снижение стоимости обработки (кэширование, батчинг).

## Q3 — Публикации
- Публичные страницы гайдов.
- Коллекции/теги.
- Расширенный экспорт (DOCX/EPUB).

## Q4 — Collaboration
- Совместное редактирование.
- Роли и рабочие пространства.
- Шаблоны гайдов.
