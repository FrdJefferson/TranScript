# Scope

## In Scope (MVP)
- YouTube URL → обработка → гайд.
- ASR с таймкодами (Whisper/Vosk или внешний API).
- Шаги и иллюстрации.
- Редактор текста и кадров.
- Экспорт MD/PDF.
- Шаринг read-only по ссылке.

## Out of Scope (MVP)
- Совместное редактирование.
- Мульти-язычный UI.
- Полноценная система ролей/организаций.
- Публичные каталоги гайдов.
- Встраивание в сторонние платформы.

## Assumptions
- У пользователя есть право на обработку видео.
- Для длинных видео требуются лимиты и квоты.
