# Testing

## 1. Unit
- Сегментация и нормализация текста.
- Форматирование шагов.
- Экспорт (MD/PDF) на малых примерах.

## 2. Интеграция
- Пайплайн без внешних провайдеров (локальные мок-сервисы).
- Очереди и ретраи.
- Сохранение статусов и прогресса.

## 3. E2E
- URL YouTube → гайд → редактирование → экспорт.
- Ошибки лимитов (длина видео, квоты).
- Шаринг по ссылке (read-only).

## 4. Наблюдаемость
- Проверка `traceId` по цепочке воркеров.
- Алерты на рост `failed` и таймауты.
